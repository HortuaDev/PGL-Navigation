## Descripción del proyecto

**PGL-Navigation** es una aplicación móvil desarrollada con **Expo** y **TypeScript**, diseñada para demostrar la implementación de **navegación avanzada** utilizando **Drawer Navigation** y **Tab Navigation** a través de **Expo Router**.

El proyecto integra funcionalidades de prácticas anteriores, mostrando un **portfolio interactivo**, una **pantalla de bienvenida**, y una **lista de recuento** como ejemplo de reutilización de componentes.

---

## Objetivos principales

- Practicar la navegación por archivos con **Expo Router**.
- Implementar un **Drawer** como menú principal.
- Integrar **Tabs** dentro del portfolio para navegación interna.
- Reutilizar componentes y pantallas de prácticas anteriores.
- Mantener un flujo de trabajo profesional en **GitHub** con ramas y commits claros.

---

## Estructura de navegación

### Drawer Navigation

- **Pantalla de bienvenida** _(pantalla inicial)_
- **Portfolio** (contiene Tabs)

### Tab Navigation (dentro del Portfolio)

- **Hobbies**
- **Código QR**
- **Lista de recuento**

### Cabecera del Portfolio

- Imagen
- Breve descripción
- Visible en todas las pestañas del Portfolio

## Cómo ejecutar la aplicación

1. Clonar el repositorio:

```
git clone https://github.com/HortuaDev/PGL-Navigation.git
```

2. Acceder a la carpeta del proyecto:

```
cd PGL-Navigation
```

3. Instalar dependencias:

```
npm install
```

4. Iniciar proyecto:

```
npx expo start
```
